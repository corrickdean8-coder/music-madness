# Music Madness

A Duolingo-style interactive music theory learning app for Mandolin, Guitar, and Banjo players.

## Features

- **18 Lessons** (9 free, 9 Pro) across Guitar, Banjo, and Mandolin
- **Interactive Fretboard** with Web Audio API note playback
- **Chord Library** with SVG diagrams and audio playback
- **Ear Training** with interval, chord quality, and scale ID exercises
- **Circle of Fifths** interactive visualization
- **Practice Tracker** with XP and streak system
- **Song Library** with 15 folk/bluegrass songs
- **Gamification** with XP, levels, and achievement badges
- **Pro Subscription** via PayPal ($9.99/mo or $79.99/yr)

## Tech Stack

- React + Vite
- Tailwind CSS
- lucide-react icons
- Web Audio API (no external audio deps)
- localStorage (no backend)

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:5173
